1 Getting Started with Emacs

First, let's open Emacs and create a new file for our calculator. In Emacs, C-x C-f means "find file" (hold Ctrl, press x, then hold Ctrl, press f). We'll create calculator.el — the .el extension means Emacs Lisp.

2 Define Calculator State

Every calculator needs to remember things: the current display, the stored operand, and which operation was chosen. In Emacs Lisp, we use defvar to define variables. Variables defined with defvar are special (dynamically scoped) and perfect for our global calculator state.

Emacs Lisp Tip: Parentheses are everything in Lisp! Every expression is wrapped in (). The first element is always the function/macro name, and the rest are arguments. For example: (+ 1 2) adds 1 and 2.

3 Create the Display Function

Now we need a function to refresh the calculator's display. In Emacs, we render our UI into a buffer — a region of text that Emacs manages. We use defun to define functions. Our display function will clear the buffer and redraw the calculator face.

Emacs Lisp Tip: defun defines a function: (defun name (args) "docstring" body). The interactive declaration makes a function callable via M-x.

5 Add Arithmetic Operations

Now the brain of our calculator! When the user presses +, -, *, or /, we store the current number as the first operand and remember which operation was chosen. The calc-operate function handles this, and calc-equals performs the actual computation using pcase for pattern matching.

Emacs Lisp Tip: pcase is Emacs Lisp's pattern matching — like a super-powered switch statement. string-to-number converts a string like "42" to the number 42.

6 Create the Keymap & Major Mode

Emacs is all about keymaps! We'll create a major mode called calc-mode with its own keymap that binds digit keys to our number function and operator keys to our operation functions. This is the magic that makes Emacs so extensible — you define the rules for how keys behave.

Emacs Lisp Tip: A major mode is how Emacs customizes behavior for a buffer. Each buffer has exactly one major mode. define-derived-mode creates a new mode inheriting from an existing one.
