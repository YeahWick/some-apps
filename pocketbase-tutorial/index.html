PocketBase Todo App

Build a complete CRUD todo list using PocketBase — an open-source backend in a single file. Interactive walkthrough!

📦 What is PocketBase?

PocketBase is an open-source backend that ships as a single executable file. It gives you an embedded SQLite database, realtime subscriptions, built-in auth, a dashboard UI, and a simple REST API — all in one ~15MB binary.

Why PocketBase? Zero dependencies. No Docker, no database server, no config files. Download one file, run it, and you have a full backend with admin dashboard. Perfect for prototypes, side projects, and learning.

🗃 Create the Todos Collection

In PocketBase, collections are like database tables and records are rows. We need a "todos" collection to store our todo items.

How? Open the dashboard at http://127.0.0.1:8090/_/, click "New collection", name it todos, and add the fields below. Or use the API — click the button to see the schema.

🔌 Install the JavaScript SDK

PocketBase provides official SDKs. The JavaScript SDK works in browsers, Node.js, and React Native.

terminal
-- Click "Install SDK" to set up the client

➕ Create Todos

Use pb.collection('todos').create() to add new records. Each call sends a POST request to the API.

javascript
const record = await pb.collection('todos').create({ title: 'Buy groceries', completed: false, }); // record.id -> "abc123def456789" // record.title -> "Buy groceries"

    📖 Read Todos

    Fetch records with getFullList(), getList() for pagination, or getOne() for a single record.

    javascript
    // Get all todos, newest first const todos = await pb.collection('todos').getFullList({ sort: '-created', }); // Get a single todo by ID const todo = await pb.collection('todos').getOne('RECORD_ID'); // Filter: only incomplete todos const pending = await pb.collection('todos').getList(1, 50, { filter: 'completed = false', });
    api-response
    -- Click "Fetch All" to read todos from the collection

    ✎ Update Todos

    Use pb.collection('todos').update(id, data) to modify a record. This sends a PATCH request.

    javascript
    const updated = await pb.collection('todos').update('RECORD_ID', { completed: true, }); // Toggles the completed status of a todo

    Click the checkmark to toggle todos you created above:

      🗑 Delete Todos

      Use pb.collection('todos').delete(id) to remove a record. Returns 204 No Content on success.

      javascript
      await pb.collection('todos').delete('RECORD_ID'); // Response: 204 No Content // The record is permanently removed
